class Ex01{}
class = 자바에서 저장할 수 있는 최소단위
-class 이름은 파일 저장이름이랑 같아야됨
-class 이름의 첫글자는 항상 대문자
-class는 메서드와 필드로 구성되어 있음
-필드 : 상태값 = 데이터, 아는것
-메서드 : 동작 = 하는것
-필드 = 변수와 상수로 나뉨
-언제든지 변할 수 있는 상태값 = 변수
-절대 변하지않는 상태값 = 상수


public static void main(String{} args){}
- () = 메서드라고 부름 ( 동작, 작업 )
- () 앞에는 이름이 나옴
- main 메서드이며 JVM이 호출해서 사용하는 메서드임
- 프로그램을 시작시키는 시작점
- 모든 프로그램에는 main 메서드가 하나만 존재하며 없어서도 안되고 복수 불가능
- main 메서드는 여러개 불가능하다 항상 1개 

System.out.println("자바의 꿈");
- System.out = 모니터
- System.in = 키보드
- out 표준 출력장치 (모니터)
- in 표준 입력장치 (키보드)
- println(" ") = 내용 출력
- " = string

print = 줄바꿈 없음
println = 줄바꿈 있음
